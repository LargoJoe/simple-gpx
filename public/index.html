Simple GPX

You can:

  • Simplify Tracks - for navigation
    • Removes unnecessary trackpoints (based on accuracy chosen) and strips out bloat
    • Split - by distance
    • Split - by Waypoint
  • Simplify Tracklogs - for retention
    • Retain timestamps
    • Retain elevation
  • Create Routes - from Tracks
  • Add Proximity Alarms - to Waypoints
Result is returned as zip file containing GPX and statistics stats.txt file. No details of tracks processed are kept on the server with everything done in memory.
