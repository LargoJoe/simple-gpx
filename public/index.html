Simple GPX

Created by Phil Whitehurst cloned by Joe Cain

You can:

  • Simplify Tracks - for navigation
    • Removes unnecessary trackpoints (based on accuracy chosen) and strips out bloat
    • Split - by distance
    • Split - by Waypoint
  • Simplify Tracklogs - for retention
    • Retain timestamps
    • Retain elevation
  • Create Routes - from Tracks
  • Add Proximity Alarms - to Waypoints
  • Add Distance Based Waypoints With Proximity Alrams
Result is returned as zip file containing a GPX file per track and gpx file for waypoints plus statistics stats.txt file. No details of tracks processed are kept on the server with everything done in memory.
