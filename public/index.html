Simplify GPX

Select a GPX to be simplified

This application will simplify your GPX tracks to reduce the file size. It will use a combination of stripping out the height data and using the Ramer–Douglas–Peucker algorithm to reduce the number of track points to a tolerance you decide. The resulting file is returned as a zipped file.

How close do you wish the new track to align to the old track? A value of 10 metres gives a good compromise between size of resulting file and how well new track follows original.
