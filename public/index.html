Simple GPX

You can:

  • Simplify Tracks - for navigation
    • Split - by distance
    • Split - by Waypoint
  • Simplify Tracks - retain timestamps, elevation for tracklogs
  • Create Routes - from Tracks
  • Add Proximity Alarms - to Waypoints

Result is returned as zip file containing GPX and stats.txt file.

No details of tracks processed are kept on the server with everything done in memory.
